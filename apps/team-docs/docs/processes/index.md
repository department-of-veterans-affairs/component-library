# Team Processes

This section documents team processes, workflows, and standard operating procedures.

## Creating a New Process Document

Click **Edit Content** in the top navigation or go directly to [/admin/#/collections/process](/admin/#/collections/process) to create a new process document.

## Process Categories

- **Development workflows** - Code review, deployment, testing
- **Design workflows** - Design critiques, handoffs, Figma processes
- **Communication** - Slack channels, meeting cadence, escalation paths
- **Documentation** - How to document components, patterns, and decisions
