# Back to top

The "Back to top" button will appear after the user has scrolled partway down the page, and clicking it will take them to the top

The _actual_ BTT component consists of the following - the rest is just context:

```html
  <div id="back-to-top-container">
    <div class="usa-content">
      <button type="button" className="usa-button">
        <span>
          <i aria-hidden="true" class="fas fa-arrow-up" role="img"></i>
        </span>
        <span>Back to top</span>
      </button>
    </div>
  </div>
```
