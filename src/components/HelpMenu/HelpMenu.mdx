### Code:

#### Example State
```
state = {
  open: false
}
```

#### Code Sample
```javascript
import HelpMenu from '@department-of-veterans-affairs/component-library/HelpMenu'

<div style={{ backgroundColor: '#112e51', width: '85px' }}>
  <HelpMenu
    cssClass='va-dropdown'
    isOpen={this.state.open}
    clickHandler={() => this.setState({ open: !this.state.open })}>
  </HelpMenu>
</div>
```
