### Code:

#### Example State
```
state = {
  page: 2,
}
```

#### Code Sample
```javascript
import Pagination from '@department-of-veterans-affairs/component-library/Pagination'

<Pagination
  className="custom-class"
  onPageSelect={(page) => this.setState({ page })}
  page={this.state.page}
  pages={3}
  maxPageListLength={3}
  showLastPage={true}
/>
```
